# Wanderlust – Airbnb Clone
Wanderlust is a full-stack web application inspired by Airbnb that allows users to explore, create, update, and delete rental listings such as hotels, bungalows, and rooms.
<br>

## Features

 ### View listings
-Browse all available rental properties.

 ### Create Listing 
– Add new hotels, rooms, or bungalows
 ### Edit Listing
 – Update details of existing listings
- ### Delete Listing 
– Remove listings easily
- ### CRUD Operations 
– Fully functional backend with RESTful APIs

## Tech Stack

### Frontend:
- HTML
- CSS
- JavaScript
- EJS

### Backend:
-Node.js
-Express.js

### Database:
-MongoDB
